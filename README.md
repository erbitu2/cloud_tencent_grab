# 腾讯云服务器秒杀抢购工具
腾讯云秒杀活动抢购。支持指定地域

## 功能特点

- 多地域分布抢购：支持华北、华东、华南同时抢购\(支持指定地域\)

- 自动等待秒杀：到点自动发起抢购

## 环境要求

- Python 3\.12\+

- Windows / macOS / Linux

## 安装步骤

### 1\. 安装依赖

```Plain Text
pip install requests
```

### 2\. 获取登录Cookie

手动从腾讯云网站获取：登录网站后F12通过控制台执行 document\.cookie 命令获取

\!\[Cookie获取图片\]\(此处替换为你的图片地址\)
